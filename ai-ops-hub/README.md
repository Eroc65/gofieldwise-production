# AI Operations Hub

This project is an AI-powered operations hub with a control agent and specialist workers for product, research, browser automation, social posting, and ad ops.

## Structure
- `/apps/console` — Next.js operator dashboard
- `/apps/api` — FastAPI or webhook service
- `/workers/` — Specialized agent workers
- `/packages/` — Shared code, integrations, and LLM orchestration
- `/supabase/` — Database, migrations, and edge functions
- `/docs/` — Architecture, runbooks, policies

See docs for implementation details and runbooks.
