# Policies

## Approval Required For
- Production deployments
- Ad budget increases
- Public posts
- External browser form submissions
- Destructive DB changes
- Credential or billing changes

## Auto-Run Allowed For
- Research
- Draft generation
- QA screenshots
- Internal summaries
- Read-only reporting
- Preview deployments

## Security
- No sensitive keys in the browser
- All risky actions logged and auditable
- Approval decisions stored permanently
